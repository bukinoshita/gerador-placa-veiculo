# gerador-placa-veiculo

> Gerador brasileiro de Placa de Veículo

## Install

```
$ yarn add gerador-placa-veiculo
```

## Usage

```ts
import { geradorPlacaVeiculo } from 'gerador-placa-veiculo'

geradorPlacaVeiculo()
// ~> HDO1T27
```

## API

This module follows the new Mercosur license plates

### geradorEstadoCivil()

Returns a `string`
